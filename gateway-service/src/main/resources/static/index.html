This is a lightweight landing page for the API gateway. If you see this page, the gateway is reachable and healthy enough to serve static assets.

Catalogue API

Commande API

Paiement API

Route through gateway to payment endpoints.

GET /api/v1/paiements

Quick health checks

Use these commands in a shell:

curl http://localhost:8080/actuator/health
curl http://localhost:8080/api/v1/produits
curl http://localhost:8080/api/v1/commandes
curl http://localhost:8080/api/v1/paiements
Tip: a 404 on route links can mean the downstream service has no data yet, not that the gateway is down.
